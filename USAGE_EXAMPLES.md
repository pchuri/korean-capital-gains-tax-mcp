# 📚 Claude Desktop 사용 예시

이 문서는 Claude Desktop에서 Korean Capital Gains Tax MCP Server를 사용하는 실제 예시들을 제공합니다.

## 🏠 기본 양도소득세 계산

### 1세대 1주택 일반적인 경우

**사용자 입력:**
```
서울 강남구 아파트를 8억원에 2017년 1월에 샀고, 2024년 12월에 15억원에 팔았습니다. 
7년간 거주했고, 중개수수료 800만원이 들었습니다. 
1세대 1주택입니다. 양도소득세를 계산해주세요.
```

**Claude 응답 예시:**
```
양도소득세 계산 결과:

💰 최종 납부세액: 45,750,000원
📊 양도차익: 682,000,000원  
📈 과세표준: 457,500,000원
🎯 적용세율: 10% (1세대1주택 우대)

📋 계산 단계:
1. 양도차익: 1,500,000,000 - 800,000,000 - 18,000,000 = 682,000,000원
2. 장기보유특별공제 (7년, 거주): 682,000,000 × 36% = 245,520,000원
3. 양도소득금액: 682,000,000 - 245,520,000 = 436,480,000원
4. 과세표준: 436,480,000 - 2,500,000 = 433,980,000원
5. 산출세액: 누진세율 적용

💡 1세대1주택 12억원 비과세 한도 초과로 일부 과세
✅ 7년 보유 + 거주요건으로 36% 장기보유특별공제 적용
```

## 🏘️ 다주택자 중과세 계산

**사용자 입력:**
```
조정대상지역에서 3주택을 보유한 다주택자입니다.
2019년에 6억원에 산 아파트를 2024년에 9억원에 팔았습니다.
중과세율을 포함해서 계산해주세요.
```

**Claude 응답 예시:**
```
다주택자 중과세 계산 결과:

💰 최종 납부세액: 145,200,000원
📊 양도차익: 282,000,000원
📈 과세표준: 262,500,000원  
🎯 적용세율: 54% (중과세 적용)

⚠️ 중과세 사유:
- 조정대상지역 내 3주택 이상 보유
- 기본세율 24% + 중과 30% = 54%
- 보유기간 5년으로 장기보유특별공제 16% 적용

📊 세율 구성:
- 기본세율: 24%
- 다주택자 가산: +30%
- 최종세율: 54%
```

## 🔍 데이터 검증

**사용자 입력:**
```
다음 정보가 올바른지 검증해주세요:
- 주택, 취득가 -5억원, 취득일 2020년 13월
- 양도가 7억원, 양도일 2019년 1월
- 1세대1주택
```

**Claude 응답 예시:**
```
❌ 입력 데이터에 오류가 있습니다

오류 목록:
1. 취득가액: -5억원은 유효하지 않습니다 (0보다 큰 값이어야 함)
2. 취득일자: "2020년 13월"은 잘못된 날짜 형식입니다 (YYYY-MM-DD 형식 필요)
3. 날짜 순서: 양도일(2019-01)이 취득일(2020년)보다 빠를 수 없습니다

✅ 수정 제안:
- 취득가액: 5억원 (500,000,000원)
- 취득일자: 2020-01-15 (정확한 날짜로)
- 양도일자: 2024-01-15 (취득일 이후로)
```

## 📚 계산 과정 설명

**사용자 입력:**
```
1세대1주택 양도소득세 계산 과정을 자세히 설명해주세요.
보유기간 5년, 거주기간 3년인 경우의 세율과 공제율을 알려주세요.
```

**Claude 응답 예시:**
```
1세대1주택 양도소득세 계산 과정 상세 설명

📖 계산 공식:
1단계: 양도차익 = 양도가액 - 취득가액 - 필요경비
2단계: 양도소득금액 = 양도차익 - 장기보유특별공제  
3단계: 양도소득과세표준 = 양도소득금액 - 기본공제(250만원)
4단계: 산출세액 = 과세표준 × 세율

🎯 귀하의 경우 적용 내용:

장기보유특별공제:
- 보유기간 5년: 일반 16%, 거주가산 28%
- 거주기간 3년으로 거주요건 충족
- 적용공제율: 28% ✅

비과세 혜택:
- 1세대1주택 12억원 비과세 한도
- 보유기간 2년 이상: ✅ 충족
- 거주요건 2년 이상: ✅ 충족 (3년 거주)

세율 적용:
- 기본 누진세율 적용 (6%~45%)
- 중과세 해당없음
- 단기보유 중과세 해당없음

⚖️ 관련 법령:
- 소득세법 제88조~제108조 (양도소득세)
- 소득세법 제168조의2 (1세대1주택 비과세)
```

## 🏢 상업용 부동산 계산

**사용자 입력:**
```
상업용 건물을 계산해주세요.
2018년 10억원 취득, 2024년 18억원 양도
필요경비 5천만원입니다.
```

**Claude 응답 예시:**
```
상업용 부동산 양도소득세 계산

💰 최종 납부세액: 189,000,000원
📊 양도차익: 750,000,000원
📈 과세표준: 597,500,000원
🎯 적용세율: 누진세율 (최고 42%)

📋 상업용 부동산 특징:
❌ 1세대1주택 비과세 해당없음
✅ 장기보유특별공제 6년: 20% 적용
❌ 거주가산 해당없음

💡 참고사항:
- 상업용 부동산은 주거용과 달리 중과세 면제
- 임대수익이 있었다면 별도 신고 필요
- 부가가치세 과세 대상 여부 확인 필요
```

## 🏡 단기보유 중과세

**사용자 입력:**
```
작년에 산 집을 올해 팔려고 합니다.
2023년 3월에 8억원, 2024년 8월에 12억원
단기보유 중과세가 얼마나 될까요?
```

**Claude 응답 예시:**
```
⚠️ 단기보유 중과세 적용

💰 최종 납부세액: 158,000,000원
📊 양도차익: 382,000,000원
📈 과세표준: 379,500,000원
🎯 적용세율: 40% (단기보유 중과세)

🚨 중과세 사유:
- 보유기간: 1년 5개월 (2년 미만)
- 단기보유 중과세율 40% 적용
- 장기보유특별공제 해당없음 (3년 미만)

⏰ 절세 방법:
- 2025년 3월 이후 양도 시 중과세 면제
- 2026년 3월 이후 양도 시 장기보유특별공제 적용

💡 참고:
현재 양도 시 세액이 매우 높으니, 가능하다면 
2년 이상 보유 후 양도를 권장드립니다.
```

## 📞 문제 해결

**MCP 연결 문제 시:**
```
MCP 서버 연결에 문제가 있나요? 다음을 확인해주세요:
1. 설정 파일 경로가 정확한지
2. 서버가 정상적으로 빌드되었는지  
3. Claude Desktop이 재시작되었는지
```

**계산 오류 시:**
```
계산 결과가 이상해 보이는데, 데이터를 다시 검증해주세요.
특히 다음 항목들을 확인해주세요:
- 날짜 형식 (YYYY-MM-DD)
- 금액의 단위
- 조정대상지역 여부
```
